# Linux Basics
## Commands 

- ### Directories
    - **pwd** - Display current directory path
    ```bash
    $ pwd
    ```
    - **cd** - Change directory
    ```bash
    $ cd <name>
    ```
    - **cd..** - Go to parent directory
    ```bash
    $ cd ..
    ```
    - **ls** - List the contents of directory
    ```bash
    $ ls
    ```
